# Stats

L'objet <mark style="color:$primary;">Stats</mark> contient les caractéristiques du combattant.

Vous pouvez les obtenir de cette manière :&#x20;

<pre class="language-lua"><code class="lang-lua"><strong>local agility = myFighter.stats[enum_Stat.AGILITY]
</strong><strong>console.print("Agilité: "..agility)
</strong></code></pre>
