Cet outil permet de générer des morceaux de codes pour l'utilisation des méthodes :&#x20;

{% content-ref url="../methodes/pixel" %}
[pixel](https://doc.frigost.dev/abonnement-frigost-bot/methodes/pixel)
{% endcontent-ref %}

{% content-ref url="../methodes/mouse" %}
[mouse](https://doc.frigost.dev/abonnement-frigost-bot/methodes/mouse)
{% endcontent-ref %}

Quand <mark style="color:$primary;">PixelTool</mark> est activé, un clique droit permet d'afficher ce menu contextuel :&#x20;

<figure><img src="https://3373271050-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-Mfulp0LhbJrdkH_fl5w%2Fuploads%2F1aN5xXBwhluIzWDWHMnv%2Fimage.png?alt=media&#x26;token=e561a217-f339-4cff-bdf4-2865a13c8f54" alt=""><figcaption></figcaption></figure>

Chaque élément du menu permet de générer un petit morceau de code à coller dans votre script.&#x20;

* Par exemple, "<mark style="color:$primary;">Clique gauche</mark>" donne un code comme celui là :&#x20;

```lua
mouse.click(1154,441, false)
```

* Par exemple, "<mark style="color:$primary;">Vérifier que la couleur est</mark>" donne un code comme celui là :&#x20;

```lua
pixel.color(1311,490) == "#476466"
```
