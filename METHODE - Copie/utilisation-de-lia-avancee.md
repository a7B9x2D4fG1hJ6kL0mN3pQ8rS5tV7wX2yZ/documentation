{% hint style="info" %}
Quand l'IA avancée est activée sur un combattant, toutes les autres options (focus, type de déplacement, sorts ...) sont ignorées, c'est normal car <mark style="color:$primary;">c'est vous qui décident</mark> ce que votre personnage doit faire et non pas le logiciel.
{% endhint %}
