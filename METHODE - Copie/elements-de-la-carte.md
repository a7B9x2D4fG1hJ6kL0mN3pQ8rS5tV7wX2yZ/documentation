Cet outil permet de générer des morceaux de codes et de récupérer certains identifiants :&#x20;

<figure><img src="https://3373271050-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-Mfulp0LhbJrdkH_fl5w%2Fuploads%2FNHXbMHYs699NUR9ySaQz%2Fimage.png?alt=media&#x26;token=9ddaf27c-43a9-4b78-af0e-bd2b9decfd2b" alt="" width="563"><figcaption></figcaption></figure>

Chaque élément du menu permet de générer un petit morceau de code à coller dans votre script.&#x20;

* Par exemple, "<mark style="color:$primary;">Copier le chemin vers le haut</mark>" donne un code comme celui là :&#x20;

```lua
{ map = "191104000", path = "top" }, -- [4,-19]
```

* Par exemple, "<mark style="color:$primary;">Copier le chemin vers l'interactif</mark>" donne un code comme celui là :&#x20;

```lua
{ map = "191104000", door = 313 }, -- [4,-19]
```
